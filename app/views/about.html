About the Stock Trading Backtest Tool

The stock trading backtest application is a client/server application with a AngularJS single page web application on the client side and a Java Spring REST application on the server side.

A listing of each of the technologies used in the development of this application is shown below.

Client Side Application

Technology Description
Backtest Client Side Source Code Custom code developed for the backtesting client side application that is version controlled on GitHub
Javascript Dynamically typed object oriented language that is supported in all common web browsers
AngularJS Javascript MV* framework that makes building dynamic single page web applications much easier
HTML Markup language for describing web documents
CSS Cascading Style Sheets is used to describe how (color, format, spacing, size, etc)HTML elements are to be displayed on the screen
Bootstrap The most popular HTML, CSS, and JavaScript framework for developing responsive, mobile-first web sites
Underscore.js A javascript library that provides many useful programming utilities
Yeoman A great scaffolding tool that saves a lot of time when starting a new web app
Bower Package manager tool that automates the process of fetching and installing third party packages
Grunt Task runner that automates things like building, testing, and minification
WebStorm Great Javascript IDE with many features that make web development much easier

Server Side Application

Technology Description
Backtest Server Side Source Code Custom code developed for the backtesting server side application that is version controlled on GitHub
Java 8 Powerful object oriented language that uses a virtual machine to enable 'write once run anywhere' capability.
Spring Framework that handles all of the "plumbing" for assembling a Java project. In addition to providing dependency injection this framework support many support projects , some of which are used in this application and cited below.
Spring Boot Greatly simplifies the creation of a Spring application by automatically handling almost all configuration requirements.
Spring REST Spring REST provides support for making a stateless web service to better create distributed systems
JSON (JavaScript Object Notation) is a name/value data-interchange format that is used to transfer information from server to clients.
Jackson Mostly runs behind the scenes to automatically convert JSON to Java objects and vice versa.
Maven Manages the projects dependencies and builds the project
YahooFinanceAPI All of the stock trading code was custom built for this application, but you have to get the historical quote data from somewhere. Yahoo Finance API is a web service that provides historical quote data upon request
YAML Used to more easily configure development and production profiles
Tomcat Open source web server which pretty much runs behind the scenes with minimal configuration thanks to Spring Boot
JUnit Java unit testing framework
Mockito Mocking framework for unit tests in Java
Spring Tools Suite Eclipse based IDE that is a must have for developing Java Spring based applications.

Cloud Deployment

Technology Description
Openshift Openshift offers a free version of their PaaS service that was more than capable of handling the cloud deployment for the Backtest application
Putty Free ssh client that is useful for connecting to servers like Openshift
